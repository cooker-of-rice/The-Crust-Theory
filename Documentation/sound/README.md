# The Crust Theory - Audio Assets (Fáze 1)

Tento dokument obsahuje přehled prvních zvukových assetů (SFX) vytvořených pro prototyp hry The Crust Theory (2D Noir Metroidvania). 

Vzhledem k deadlinu odevzdáváme první iteraci zvuků zaměřenou na pohyb hráče a základní útoky bosse. Rozšířené zvuky prostředí budou dodány v dalším updatu.

## Dodané soubory
Zvukař dodal první sadu náběrů (zatím v surovém formátu `.MOV`, pro integraci do Godot 4 budou následně konvertovány do `.OGG` / `.WAV`).

**Pohyb hráče (Foley):**
* `asfalt_walk.MOV` - Chůze po ulici / základní povrch.
* `metal_walk.MOV` - Chůze po kovových mřížích (připraveno pro level stoky).
* `wood_walk.MOV` - Chůze po dřevěných parketách (byt detektiva).
* `jump.MOV` - Odraz při skoku.
* `landing.MOV` - Těžký dopad na zem.
* `wall_slide.MOV` - Skluzavý zvuk tření o stěnu (pro mechaniku Wall Slide).

**Nepřátelé a Boss (Krabí mutanti):**
* `click.MOV` - Rychlé sklapnutí klepeta (základní útok bosse).
* `slam.MOV` - Těžký náraz / drtivý útok bosse.

## Co chybí (Plán do další fáze)
Pro kompletní audio kulisu prototypu se aktuálně pracuje na:
* Výstřel z pistole.
* Zvuky tekoucí vody a padajících kamínků (ambient do stok).
* Skřípavé zvuky blížícího se nepřítele (body-horror krabí mutanti).
* Podkresová jazzová hudba.
