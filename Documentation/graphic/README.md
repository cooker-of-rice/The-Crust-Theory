# The Crust Theory - Grafika
>autoři-Bican, Brož

Tady je info k vizuálu a nástrojům pro hru (The Crust Theory - 2D noir detektivka).

## Software (co používám)
Grafiku dělám kombinovaným procesem, abych ušetřil čas:

1.  **Malování (MS Paint)**
    * Tady dělám hrubé náčrty (blockout) a rozvržení levelů. Prostě jen čáry a obdélníky pro kompozici.
2.  **Generativní AI**
    * Náčrt z malování vezmu a nechám podle něj vygenerovat detailní pixelart (prompty řeším bokem).
3.  **Adobe Photoshop**
    * Finální úpravy. AI tam občas hází blbosti, takže to retušuju, dělám ořezy a chystám vrstvy pro Godot.

## Hardware
* Normální PC.
* Myš a klávesnice (tablet nemám, na ty náčrty v malování stačí myš).

## Výstup a vizuální styl
Cílem je vytvořit konzistentní vizuál, i když to generuje AI.
* **Žánr:** 2D Noir Metroidvania s prvky body-horroru.
* **Styl:** High-fidelity Pixelart, 70. léta, tmavé barvy a hodně stínů (noir).
* **Pravidla:**
    * Všechno je striktní 2D pohled z boku (side-view).
    * Interiéry jsou řešené jako řezy budovou (cutaway).
    * Poměr stran pozadí je většinou 21:9.
